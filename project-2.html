  • CSS
    •

    Cascading Style Sheets (CSS) is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML (including XML dialects such as SVG, MathML or XHTML). [2] CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

  • Core Java
    •

    Core Java refers to the fundamental concepts of Java used to build robust, platform-independent applications. It includes object-oriented programming principles, data types, control statements, exception handling, multithreading, and collections. Core Java forms the foundation for advanced Java technologies like JDBC, Spring, and enterprise-level application development.
